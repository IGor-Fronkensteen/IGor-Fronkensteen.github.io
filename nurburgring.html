🏁 Nürburgring Lap Time Predictor

Predicting Nordschleife lap times from raw vehicle specs using machine learning and feature engineering.

🚘 Project Overview

This project predicts lap times on the Nürburgring Nordschleife using real-world car specs. It’s a study in how physical parameters like power, weight, and aerodynamics map to real-world racetrack performance. The goal was to create an intuitive model that could not only predict, but also explain.

🛠️ Methodology

  • Data Scraping: Gathered car specs and lap times from FastestLaps.com
  • Feature Engineering: Calculated power-to-weight, log-transform of lap time, etc.
  • Modeling: Linear regression → Ridge → Random Forest
  • Evaluation: R², residuals, and SHAP values for model explainability

📊 Key Findings

The tuned Random Forest achieved an R² of 0.94 with good generalization. SHAP analysis revealed specific power and weight were the most influential features—validating core race engineering intuition.

🔍 What I Learned

  • Feature selection is the heart of predictive modeling
  • Log-transforms can linearize complex physical relationships
  • Interpretability (SHAP) makes results trustworthy and insightful
