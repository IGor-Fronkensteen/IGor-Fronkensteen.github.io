Nürburgring Lap Time Predictor

Predicting performance from raw specs using machine learning.

🚗 Overview

This project builds a predictive model for lap times on the Nürburgring Nordschleife using real-world vehicle specifications like weight, horsepower, wheelbase, and more. The goal: explore how machine learning can map engineering parameters to real-world track performance.

🧪 Approach

  • Data collection: Scraped vehicle spec and lap time data from FastestLaps.com
  • Preprocessing: Feature extraction (e.g. power-to-weight), outlier handling, log-transform of lap time
  • Modeling: Baseline Linear Regression, then Ridge/Lasso/Random Forest with hyperparameter tuning
  • Evaluation: R², residual plots, SHAP analysis for model interpretability

📊 Key Results

The best model (Random Forest with tuned hyperparameters) achieved an R² of 0.94, accurately capturing the impact of power-to-weight ratio, drag, and downforce. SHAP plots revealed that weight and specific power were the top contributors.

🔍 What I Learned

  • Feature engineering is crucial—raw horsepower isn’t as telling as power-to-weight
  • Model interpretability (e.g. SHAP) builds trust in predictive systems
  • Model performance isn’t just about accuracy—it’s about domain insight
